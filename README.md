# `@olegkorn/eslint-config`
Convenient for me eslint config.

## Installation

To install, run:

```sh
# npm
npm install @olegkorn/eslint-config --save-dev

# yarn
yarn add @olegkorn/eslint-config --dev
```

And just extend eslint config:

```js
{
  "extends": [
    // ...
    "@olegkorn"
  ]
}
```
